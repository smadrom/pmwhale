# Support

Use [GitHub Issues](https://github.com/smadrom/pmwhale/issues) for reproducible
bugs and focused feature proposals. Include the pmwhale version or commit,
operating system, Python or Bun version, command, expected result, and sanitized
error output.

This community project provides best-effort support. It cannot offer trading,
financial, legal, jurisdictional, or account-specific advice. Never post private
keys, credentials, private datasets, or claims linking a wallet to a real person.

Security vulnerabilities must follow [SECURITY.md](SECURITY.md), not a public
issue.
